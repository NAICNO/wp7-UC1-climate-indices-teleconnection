# Deployment Log

Originally developed and tested on Sigma2 HPC (Saga cluster), the NAIC Teleconnection Demonstrator now runs on NAIC Orchestrator VMs. The migration simplified the deployment workflow by removing the need for job schedulers, queue systems, and multi-hop SSH tunneling. The current setup uses standard VM provisioning through the [NAIC Orchestrator portal](https://orchestrator.naic.no), with direct SSH access and straightforward Jupyter Lab sessions.

For current deployment instructions, see the main [README.md](../README.md).
